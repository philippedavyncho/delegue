# SEO Roadmap DASMARK PHARMA

## 1) Base technique deja en place
- Domaine configure: `https://dasmarkpharma.ci`
- Balises SEO globales dans le layout
- Canonical + Open Graph + Twitter cards
- JSON-LD injecte depuis les pages
- `robots.txt` en production
- `sitemap.xml` dynamique en place

## 2) Pages SEO a creer (sans changer le design)

- `/services/marketing-pharmaceutique/`

## 3) Priorite business (ordre de production)
1. `/services/marketing-pharmaceutique/`
2. `/services/visite-medicale/`
3. `/formation/delegue-medical/`

## 4) Regles on-page minimales
- 1 H1 unique par page (mot-cle principal)
- Title < 60 caracteres
- Meta description < 155 caracteres
- 3 a 5 H2 orientes intention + conversion
- 1 CTA principal: `Parler sur WhatsApp`
- 1 CTA secondaire: `Demander un rendez-vous`

## 5) Donnees structurees a appliquer
- Home: `Organization`
- Services: `Service` + `BreadcrumbList`
- Formation: `Service` (ou `Course` si programme formalise)
- Local: `Service` + signaux geographiques

## 6) Suivi conversion SEO
- Clic WhatsApp
- Envoi formulaire
- Appel telephonique
- Source organique (Search Console + Analytics)

## 7) Controle qualite avant mise en ligne
- Pas de 404 dans le sitemap
- Canonical auto-reference sur chaque page
- Pas de duplication Title/H1
- Temps de chargement mobile < 3s
